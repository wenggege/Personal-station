  • 包括门卫、巡逻、守护等基础保安服务，公司主要业务分布在珠三角地区，服务对象涉及党政机关、公交、酒店、银行、商场、餐饮、房地产、工厂企业等，协助公安机关维护社会秩序。
  • 1.门卫保安服务：是指保安服务公司依照合同约定，派出保安员依据国家法律、法规、规章以及雇请单位的规章制度，对从雇请单位大门或门口通过的职工及外来人员、车辆等进行职责上的严格把守，具体实施验证、检查、登记，以全面维护雇请单位的内部秩序，保障人身、场所及其他事项的安全的一项保安业务。
  • 2.保安巡逻服务：是指保安服务公司依照合同约定，派出保安人员对雇请单位制定和特定目标或区域进行巡逻警戒，以维护其正常的生产、工作、教学、科研及经营等秩序，保证其安全的一项保安业务。
  • 3.场所服务型保安服务：是指保安服务公司依据保安合同规定，向某公共场所派员，依照国家法律的有关规定和场所单位内部的有关制度，对其加强安全保安服务，以防止违法犯罪和各种灾害事故的发生，维护该场所的正常社会治安秩序，确保人、财、物的绝对安全而开展的一项综合性保安业务。这里所讲的场所是指公共场所。所谓公共场所是指人员高度集中且大都没有特定关系，流动频繁，成分复杂，易发生违法犯罪活动和治安灾害事故的场所。通常这些场所包括大型集会场所、体育活动场所、商品购物场所、文化娱乐场所、参观游览场所、展览展销场所、消遣游玩场所、旅行候机场所等。这些场所人员集中、流动性大、不易控制，易发生拥挤、踩踏、打架、扒窃、赌博、诈骗、流氓滋事等非法活动和重大事件或案件。
  • 4.保安守护服务：是指保安服务公司依照合同约定，派出保安人员对保安合同所规定的特定目标、项目和内容进行专门的看守和保护，以保证其安全的一项保安业务。
