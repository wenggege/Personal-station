降血脂药又添生物药
治疗高脂血症是预防心脑血管疾病、降低死亡率的重要手段之一，目前国内降血脂药物市场竞争激烈，有包括他汀类药品在内的几十个降血脂药，还有1000个左右的相关功能保健品在市场中角逐。近日，由著名心血管病专家胡大一主持的“中国老年人心血管防治高峰论坛”中，为国内医学界又引入一个来自古巴的降血脂药PPG（多廿烷醇）。随着PPG在临床上的推广，降血脂药市场将在一次被细分。 胡大一介绍说，由古巴最高医药研究机构国家科学研究中心研制的PPG是从古巴西部的特种甘蔗中提取出来的一种物质，可以降低胆固醇，抗血小板聚集以及抑制脂质过氧化作用，主要用于治疗动脉粥样硬化性疾病。该产品1991年在古巴上市，是古巴首个纯生物降脂药，与雪茄一并被称为古巴两大“国宝”。 据了解，PPG目前已经在全球30多个国家上市，在中国，古巴国家科学研究中心和山东淄博山川药业已正式签约，已于去年11月在中国上市。 在强手如林的降血脂药物之中，PPG能预防心脑血管疾病的发生，安全性高，发挥着“血管清道夫”的作用。一些降脂药物如他汀类药物虽然降脂效果明显，但有升高血清转氨酶、肌酸激酶等不良反应，不适合长期服用。胡大一表示，PPG适合作为老年人一级预防的处方药，适宜不能长期耐受他汀类药物人群，更需要重视安全性的人群；另外，PPG也可以和其他降血脂药联合用药，在治疗脂质异常的同时，提高中老年人的生活质量。
